## Palenight

![CAPTURE!](https://raw.githubusercontent.com/pojokcodeid/pcode-theme/main/palenight-fleet.png)

- Create VSIX file

```
npm i -g vsce
cd pcode-theme
vsce package
vsce publish
```

## Untuk Pengguna Linux

- Tambahkan pada file config.json

```json
{
  "window.titleBarStyle": "custom"
}
```
